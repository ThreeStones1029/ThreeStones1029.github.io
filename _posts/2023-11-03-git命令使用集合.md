# 一、前言

本文主要记录一些常用的git命令，为了方便以后使用。前提，已经初始化过本地文件夹为仓库，也即有.git的文件夹，同时图床等图片路径已经建立。

# 二、git常用命令

## 2.1.添加文件到暂存区

~~~bash
git add filename
~~~

## 2.2.添加提交信息

~~~bash
git commit -m '提交信息'
~~~

## 2.3.查看状态

~~~bash
git status
~~~

## 2.4.提交到远程仓库

~~~bash
git push -u origin master
~~~

## 2.5.重命名

修改的是已经git管理的文件,还没有用git管理的直接修改文件名即可。

### 2.5.1.修改文件名

~~~bash
git mv old_file_name new_file_name
~~~

### 2.5.2.添加提交信息

~~~bash
git commit -m "提交信息"
~~~

### 2.5.3.提交到远程仓库

~~~bash
git push -u origin master
~~~

### 2.5.4.更新远程仓库到本地仓库

~~~bash
git pull origin master
~~~
